# 💎 Elite Study Hub: Collaborative Intelligence Platform

[![GitHub Stars](https://img.shields.io/github/stars/Saikaushik007/COLLABORATIVE-STUDY-TOOL?style=for-the-badge)](https://github.com/Saikaushik007/COLLABORATIVE-STUDY-TOOL/stargazers)

**Elite Study Hub** is a next-generation collaborative learning ecosystem designed to transform solitary studying into a high-performance, collective intelligence experience. Built for students who demand precision, security, and AI-driven insights.

---

## 🎯 The Vision
Most study tools focus on just "holding" information. Elite Study Hub focuses on **synthesizing** it. By combining real-time collaboration with advanced AI co-pilots, it bridges the gap between raw notes and deep conceptual mastery.

---

## 🚀 Key Features

### 🧠 AI Intel Co-Pilot (Powered by Gemini 2.0)
*   **Instant Synthesis**: Transform hours of session notes into concise, actionable intelligence with one click.
*   **Dynamic Assessment**: Generate tailor-made quizzes and flashcards in real-time based on the current discussion.
*   **Adaptive Explainer**: Get complex concepts broken down by a high-tier AI context-aware of your whiteboard and chat history.

### 🎨 Elite Collaboration Canvas
*   **Real-Time Vector Whiteboard**: Ultra-low latency collaborative drawing and diagramming.
*   **Unified AV Presence**: Integrated high-definition audio and video presence for "face-to-face" digital study labs.
*   **Session State Persistence**: All canvas data is saved automatically, so your "eureka moments" are never lost.

### 📊 Performance Analytics & Gamification
*   **Productivity Heatmaps**: Visualize your focus intensity and session efficiency over time.
*   **Hall of Fame**: A competitive leaderboard system rewarding high-quality contributions and study consistency.
*   **Worldwide Network**: A global timeline showing upcoming broadcasts and study marathons across the community.

## 🛠️ Technology Stack

| Layer | Technology | Purpose |
| :--- | :--- | :--- |
| **Frontend** | **Next.js 14 (App Router)** | High-performance React framework with server-side rendering. |
| **Styling** | **Glassmorphic CSS** | Custom, premium dark-mode aesthetic with fluid micro-animations. |
| **Backend** | **Node.js & Express** | Scalable REST API and WebSocket orchestration. |
| **Database** | **Prisma & SQLite** | Type-safe ORM for reliable data modeling and lightning-fast queries. |
| **AI Engine** | **Google Gemini SDK** | Multi-model fallback chain (Flash 2.0/2.5) for infinite reliability. |
| **Real-time** | **Socket.io** | Bi-directional, low-latency communication for presence and canvas updates. |

## 💻 Get Started

1.  **Clone the Repo**:
    ```bash
    git clone https://github.com/Saikaushik007/COLLABORATIVE-STUDY-TOOL.git
    ```
2.  **Install Foundations**:
    ```bash
    npm run install:full
    ```
3.  **Local Launch**:
    ```bash
    npm run dev:all
    ```

## 🛡️ Security & Privacy
Designed with confidentiality in mind. All sensitive credentials are kept strictly local via `.env.example` configurations, ensuring zero leakage of API keys or private session data to the public cloud.
